# Thư mục dữ liệu đã xử lý

Đặt các file đã làm sạch/chuyển đổi từ `data/raw/` vào đây.

Định dạng khuyến nghị:

```json
{
  "documents": [
    {
      "id": "CV-001",
      "title": "Tên văn bản",
      "doc_type": "Công văn",
      "source": "Tên nguồn hoặc URL",
      "content": "Nội dung đã làm sạch"
    }
  ]
}
```

File trong thư mục này có thể được upload trực tiếp qua tab `Kho tri thức` của app.
